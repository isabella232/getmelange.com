Melange

The new decentralized social network that puts data back in the user's control.

Get Developer Preview

for Max OS X (others)

Private by default.

Melange is a new secure and decentralized social platform. Due to its unique structure, Melange is able to ensure that users, rather than some central authority, retain control over their data.

On Melange, you control who you talk to, share data with, and how secure you are.


Extensible to the core.

Melange has support for Apps baked in. Without having to re-add all your friends, you can go from sharing 140-character updates, to sharing photos with filters, or even something completely new. Check out the App tab above to see which Apps are available. If you don't see what you want, you can build your own on the Melange Plugin API.

If plugins weren't enough, Melange was built on the same core technologies that power Atom the new editor from Github. It uses a combination of HTML, JS, and Go to perform actions. You can tinker with the look and feel since all of the source is provided with the application.

Coming soon to a computer near you.

Currently, Melange is in a Developer Preview. If you are interested in participating and building applications for our platform, go ahead and download the software. Make sure to check out the Developer Center for guides and API references.
